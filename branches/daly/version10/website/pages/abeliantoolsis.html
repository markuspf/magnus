Abelian Group Questions

Is A1 ...

There are 5 questions that Magnus can answer about an abelian group:
  • Is A1 trivial?
  • Is A1 finite?
  • Is A1 free abelian?
  • Is A1 automatic?
  • Is A1 hyperbolic?
Given a finitely generated abelian groups it can be decomposed into a direct product of cyclic groups. From this decomposition we can answer several questions.

Is A1 trivial?

It is trivial if this decomposition consists only of a single cyclic group whose order is 1.

Is A1 finite?

It is finite if there are no infinite cyclic summands and its order is the product of the orders of finite cyclic factors. Otherwise its order is infinite.

Is A1 free abelian?

If there are no finite cyclic groups occurring in the decomposition then it is free abelian.
