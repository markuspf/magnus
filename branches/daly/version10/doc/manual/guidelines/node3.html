Organisation of Code

It is essential to bear in mind that your code will have to be read and understood by other people, at first to be properly integrated into the system, and possibly later to be modified, extended, or otherwise adapted.

Therefore, one should strive to keep the implementation as modular as possible with clearly defined interfaces. Beyond this general principle, the following practices are important.
