The Object View

The FE provides a procedure named objectView for use as a viewBuilder in a viewStructure argument to the create_ command (§2.3.1.1). The viewParameters it expects has the form:

{viewID heritage}

The viewID is a positive integer which is the definitive reference shared by the FE and SM to the object view.

The heritage is Hot text which describes what the object is, and where it came from. Evaluation of heritage is delayed until the view is (re)displayed, to allow for possible name changes. Thus the heritage of a new word could be

{A word in the generators of [_name_ 29]}
