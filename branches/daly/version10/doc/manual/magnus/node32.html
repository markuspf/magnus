User class ListIterator<ListType>



What and Where

The class
template <class ListType> class ListIterator :
public ObjectOf< ListIteratorRep<ListType::ListElementType> >
is defined in magnus/back_end/general/include/List.h.

It is for iterating over lists.



Special Notes

A Copy constructor, assignment operator, and destructor are supplied by the compiler.

Copying or assigning an iterator is guaranteed to produce an iterator which visits the (rest of the) same elements in the same order as the original.

All iterators work the same way. See §2.1.2.11 for examples.
