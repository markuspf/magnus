User class AssociationsIterator<Key,Val>



What and Where

The class
template <class Key, class Val> class AssociationsIterator :
public ObjectOf< AssociationsIteratorRep<Key,Val> >
is defined in magnus/back_end/general/include/Associations.h.
It is for iterating over an AssociationsOf<Key,Val>.



Special Notes

  • A copy constructor, assignment operator, and destructor are supplied by the compiler.

  • Copying or assigning an iterator is guaranteed to produce an iterator which visits the (rest of the) same elements in the same order as the original.

  • All iterators work the same way. See §2.1.2.11 for examples.
