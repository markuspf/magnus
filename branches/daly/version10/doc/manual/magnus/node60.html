Predefined Procedures

The FE provides a number of procedures which can be used as action arguments in menu definitions (§2.3.1.3). If procedure _foo_ takes arguments bar, , bar, then the action must be written

{_foo_ bar...bar}.

The arguments are not keyword arguments. Each of these procedures has the side-effect that when they exit, they have defined certain variables for use as substrings in the corresponding message argument of the realItem in the menu definition. The names of the variables are listed with each procedure.
