User class VectorOf<T>



What and Where

The class
template <class T> class VectorOf : public ObjectOf< VectorRep<T> >
is defined in magnus/back_end/general/include/Vector.h.

A VectorOf<T> is good for ordered sequences of T which change size infrequently, and for which O(1) access is desired.



Special Notes

Class T must have

  1. A default constructor
  2. A copy constructor
  3. A destructor
  4. An assignment operator
  5. An == operator

A Copy constructor, assignment operator, and destructor are supplied by the compiler.
