User class Elt



What and Where

The class
class Elt : public ObjectOf< EltRep >
is defined in magnus/back_end/Elt/include/Elt.h.
It is a pseudo-abstract class which can represent any group element.



Special Notes

A copy constructor, destructor, and assignment operator are supplied by the compiler.

Since Elt is pseudo-abstract, there is no default constructor.
