Definition Dialogs

The procedures _defineFPGroup_ title, _defineSubgroup_ title, _defineMap_ title, and _defineWord_ title each bring up a dialog in which the end user can define the named kind of object. The Hot text title, delimited by {}, says what is being asked for. Note that, e.g., _defineFPGroup_ could be used for several group types. The variables defined by each upon exit are ${name} and ${defn}.
