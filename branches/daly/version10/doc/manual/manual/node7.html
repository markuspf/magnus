User class AssociationsOf<Key,Val>



What and Where

The class
template <class Key, class Val> class AssociationsOf :
public ObjectOf< AssociationsRep<Key,Val> >
is defined in magnus/back_end/general/include/Associations.h.
It implements simple association lists.



Special Notes

Class Key must have an == operator.
Class Val must have an assignment operator.
Both must have copy constructors and destructors.
There must be an
ostream& operator « ( ostream&, const Type& )
and a conversion of Key to Type.

Note that if Key or Val is not a user class then AssociationsOf<Key,Val> will perform inefficient copying operations.
