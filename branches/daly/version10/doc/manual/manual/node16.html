User class SetOf<T>



What and Where

The class
template<class T> class SetOf : public ObjectOf< SetData<T> >
is defined in magnus/back_end/general/include/Set.h.

A SetOf<T> is good for an unordered collection of T. It features fast access and modification.



Special Notes

Class T must have

  1. An assignment operator
  2. A copy constructor
  3. An == operator
  4. A destructor
  5. A method int hash() const which need not return positive ints

A Copy constructor, assignment operator, and destructor are supplied by the compiler.
