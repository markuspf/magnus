User class ListOf<T>



What and Where

The class
template <class T> class ListOf : ObjectOf< ListRep<T> >
is defined in magnus/back_end/general/include/List.h.

A ListOf<T> is good for ordered sequences of T which change size or composition frequently, at the cost of O(n) access.



Special Notes

Class T must have

  1. A copy constructor
  2. A destructor
  3. An assignment operator
  4. An == operator

There must be an

ostream& operator « ( ostream&, const Type& )

and a conversion of T to Type.

Some ListOf methods can safely ignore bad indices, but they post a warning if SAFETY > 0 on the assumption that the caller is probably bugged.

A Copy constructor, assignment operator, and destructor are supplied by the compiler.
