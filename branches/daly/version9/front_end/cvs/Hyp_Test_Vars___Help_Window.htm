The hypothesis testing tool for variances for a one variable data set will conduct a

hypothesis test for a single variance or single standard deviation against a given  value.  Thus it will test a null hypothesis of the form

                                               Ho: s = s0

The user must supply the desired significance level and the desired alternative. The significance level must be a number between 0 and 1.  A common choice is a 5% level or a choice of .05.
