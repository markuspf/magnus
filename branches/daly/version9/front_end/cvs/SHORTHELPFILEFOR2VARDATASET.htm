SHORT HELP FILE FOR TWO-VARIABLE DATA SETS

 

 

          A two variable data set consists of two collections of numbers.  These can either be from independent populations or are paired and generally represent sets of measurements such as heights, weights, time, stress etc. from two populations. A typical example would be

 

X       3.4 ,5.6, 7.9, 8.8

Y       4.6,5.6,4.8,8.3

 

Analysis of a two variable data set differs whether they are considered from independent populations or correlated populations. If the sets of numbers are considered as samples from independent populations the two samples can be different sizes but if they are consider as samples from correlated populations they must be paired and the sample sizes must be the same.

 

CAISS-STAT allows the user to enter a two-variable data set in several different ways.  These are

     From the screen – string or list format

     From the screen – spreadsheet format

     From an existing file

     From the CAISS-STAT database
