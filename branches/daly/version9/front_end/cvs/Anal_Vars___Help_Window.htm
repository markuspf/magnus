The  Analysis of the Variance Tool will perform two standard inference procedures for the population variance or population standard deviation of the population from which the given one variable data set has been drawn.  The first procedure is to construct a confidence interval for the population variance from the sample  standard deviation of the given one variable data set. The second procedure is to evaluate a hypothesis test testing a user supplied null hypothesis (given in the form of a target variance or standard deviaiton) against a user supplied alternative hypothesis.
