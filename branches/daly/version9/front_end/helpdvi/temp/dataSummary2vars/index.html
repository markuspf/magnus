THE DATA SUMMARY TOOL


The Data Summary Tool will construct a frequency distribution for each of the two given data sets. Recall that a frequency distribution is a tabular summary of the data that arranges the data into classes and then gives a count or frequency in each class. The classes will either be constructed automatically or by user prompts.
