THE GOODNESS OF FIT TOOL


The Goodness of Fit Tool will conduct a test to determine whether a one variable data set fits a given theoretical distribution with a presecribed level of significance. The user can choose the level and signficance and test the one variable data set against a normal distirbution, an exponential distribution or a uniform distribution. The test for the normal distribution uses Geary's Test for Normality while the tests for the exponential and uniform uses standard chi-square goodness of fit methods.
