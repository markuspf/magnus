DESCRIPTION OF TOOLS FOR TWO VARIABLE DATA SETS

 

MAGNUSSTAT has the following tools for use with a two-variable data set.  It also has the ability to generate random samples from the given two variable data set. The two variable data set tools are subdivided into tools for independent populations and tools for correlated populations. The tools for correlated populations are listed under regression analysis.  In this case of correlated populations and regression analysis the sample size from each population must be the same. The available tools are:

 

     Independent populations

Full Data Analysis

Data Summaries

Graphical Analysis

Goodness of Fit

Comparison of Means

Comparison of Variances

Non-parametric Tests

Report

   Regression Analysis

            Full Regression Analysis

Graphical Analysis

Prediction and Parameter Testing

Residual Analysis

Alternate Models

Correlation Analysis

Report

 

Independent Population Tools

 

The Data Summary Tool will construct a frequency distribution for each of the two given data sets. Recall that a frequency distribution is a tabular summary of the data that arranges the data into classes and then gives a count or frequency in each class. The classes will either be constructed automatically or by user prompts.  

 

The Full Data Analysis Tool will calculate all the basic descriptive measures for each of the two given data sets.  The measures included are the mean, variance, standard deviation, range, median, first quartile, third quartile and measure of skewness.   Clicking on any of these objects will give a description of the statistical measure.

 

The Graphical Analysis Tool will construct various graphical presentations for each of the two given data sets. These will include a histogram, a frequency polygon and a normal distribution with the given mean and standard deviation superimposed on the histogram. Clicking on any of the links will give a more complete description.

 

The Comparison of Means Tool will calculate either a confidence interval for the difference of the two population means or evaluate a hypothesis test for the difference of the two population means against a given target difference.  A target difference of zero is an equality of means test.

 

The Comparison of Variances Tool will evaluate a hypothesis test for equality of the two population variances.

 

The Goodness of Fit Tool will test each of the two given data sets to see how well they fits various theoretical populations including a normal, exponential and uniform.

 

The Non-parametric test tool  will compare the given populations using the non-parametric Kruskal-Wallis Test

 

Regression Analysis Tools

 

The Full Regression Analysis Tool will compute the standard regression diagnostics. These will include the regression parameters, that is the regression slope and regression intercept for the fitted regression line; the standard error of regression,the coefficient of determination and the correlation coefficient. Clicking on any of the links will give a more complete description.

 

The Graphical Analysis Tool will plot the given data points and then superimpose the fitted regression line. 

 

The Prediction and Parameter Testing Tool Clicking on any of the links will give a more complete description. Will allow the user to do various tests concerning the regression slope and regression intercept.  It will also allow the user to construct prediction intervals based on the fitted regression model.  Finally it will allow the user to do a test of model fit. 

 

The Residual Analysis Tool will construct a graph of the sample residuals and test the mean against zero and the fit against a normal model.

 

The Alternate Models Tool will allow the user to construct various alternative one variable models.  These include exponential and power models.  The fitted coefficient of determination will be compared to that of the fitted linear model.

 

The Correlation Analysis Tool  will compute the correlation coefficient and then allow the user to construct either a confidence interval for the correlation coefficient or conduct a hypothesis test concerning the correlation coefficient.
