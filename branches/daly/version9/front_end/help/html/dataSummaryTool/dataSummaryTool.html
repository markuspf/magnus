THE DATA SUMMARY TOOL


The Data Summary Tool will construct a frequency distribution of the given one variable data set. A frequency distribution is a tabular summary of the data that arranges the data into classes and then gives a count or frequency in each class. In the data summary tool the frequency distribution will be constructed automatically with eight equally spaced classes beginning with the minimum value and proceeding to the maximum value. The user can change these settings in the data summary output box to use whatver classes are desired.

The first step in analyzing a collection of data is to place the data into some sort of summary form. This will allow the user to make some preliminary sense of what is usually presented as just a list of numbers. With statistical data the most commonly used summary form is called a frequency distribution. A frequency distribution is a table which arranges the data into classes or ranges and then provides a count or tally for each class. Hence a frequency distribution arranges the data into classes and then presents the count or frequency in each class. Data that has been summarized into a frequency distribution is called grouped data. Notice that in forming this frequency distribution we have gained conciseness. However in forming the table we lose the knowledge of the individual measurements. Beyond this, the frequency distribution gives us a picture of how the data set is distributed and hence a sample picture of the population from which the data was drawn. This will help us to construct a mathematical model for the population .

The construction of a frequency distribution is relatively straightforward. We decide on the classes we want and then count the total in each of the chosen classes. We attempt to choose the classes in such a way as to give us as much information as possible. Some guidelines to follow are :


(1) Do not use too many classes and do not use too few. In the former case we don't gain enough conciseness while in the latter we lose too much information. As a rule of thumb in practice, if possible we try to stay between 5 and 15 classes.

(2) The classes should be disjoint and cover all the data

(3) If possible the classes should have the same length.


We now introduce some terminology that is associated with frequency distributions. First, for any class its class frequency is the total for that class. For example in EXAMPLE 2 the class freqeuncy for the class 130-139 is 6. The class boundaries of a class are the largest and smallest values that go into that class. Since the weights of the subjects in EXAMPLE 2 are rounded off to the nearest pound the lower class boundary of the class 130-139 is 129.5 while the upper class boundary is 139.5. There is a slight distinction between these and the class limits which are the largest and smallest values for a class which appear in the distribution. The lower class limit of the class 130-139 is 130 while the upper class limit is 139. The class interval is the length of the interval the class covers and is found by taking the difference between the class boundaries. For example the class interval for each of the classes in the frequency distribution of EXAMPLE 2 is 10. Finally for any class its class mark is its midpoint. Its importance is that it can be used as a single number to represent the class. For example the class mark of the class 130-139 is 134.5.

There are certain alternative forms of frequency distributions that will also be encountered and used. A percentage distribution has the same format relative to classes as an ordinary freqeuncy distribution but lists, instead of the actual class frequencies, the percentage the class frequency is of the total. A percentage distribution is used most often when the individual class frequencies are large. A percentage distribution is also extremely important in trying to get a sense of the general distribution of the population.

Another alternative format for a frequency distribution is a cumulative less than distribution. In this format what will be listed will be the cumulative totals up to the end of the classes. It often serves as a sample version of the cumulative distribution function which is important in modeling populations.
