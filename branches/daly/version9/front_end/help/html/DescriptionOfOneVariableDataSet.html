LONG HELP FILE FOR ONE-VARIABLE DATA SETS

 

 

A one variable data set is a collection of numbers.  These numbers generally represent a set of measurements such as heights, weights, time, stress etc.  These will usually consist of a random sample from some continuous population.  A typical example would be

4.8, 8.9, 3.7, 6.6, 5.9

 

Analysis of a one variable data set usually includes determining the descriptive measures of the data, such as the mean and standard deviation and then using these calculations to infer population measures.  The tools available for use with a one variable data set include

 

Data Summary

Full Data Analysis

Analysis of the Mean

Analysis of the Variance

Goodness of Fit

Graphical Analysis

 Report

 

The Data Summary tool will construct a frequency distribution of the given data set. 

 

The Full Data Analysis Tool will calculate all the basic descriptive measures for the given data set.

 

The Graphical Analysis Tool will construct various graphical presentations of the given data set.

 

 

The Analysis of the Mean Tool will calculate either a confidence interval for the population mean or evaluate a hypothesis test against a given target mean.

 

The Analysis of the Variance Tool will calculate either a confidence interval for either the population variance or population standard deviation or evaluate a hypothesis test against a given target variance.

 

The Goodness of Fit Tool will test the data set to see how well it fits various theoretical populations including a normal, exponential and gamma.

 

The Report Tool will sequentially go through all the other tools.

 

 

 

CAISS-STAT allows the user to check-in a one variable data set in several different ways.  These include

 

From the Screen – String or List Format

From the Screen – Spreadsheet Format

From an Existing File

From the CAISS-STAT Database

 

Below we give a description of each of these options.

 

(1)   FROM THE SCREEN – STRING OR LIST FORMAT

 

          A one variable data set is a collection of numbers.  These numbers generally represent a set of measurements such as heights, weights, time, stress etc.  These will usually consist of a random sample from some continuous population.  In entering from the screen in string or list format  the user would enter the data one after another separated by commas.  For example

 

4.8, 8.9, 3.7, 6.6, 5.9

 

(2)   FROM THE SCREEN – SPREADSHEET FORMAT

 

          A one variable data set can also be inputted via a spreadsheet format. When the user chooses spreadsheet format a spreadsheet will appear.  For a one variable data set the user will input the data into the first column of the spreadsheet. For example

(3)   FROM AN EXISTING FILE

 

     The user can also import an existing file to be checked in as a CAISS-STAT

one variable data set.

 

 

(4)   FROM THE CAISS-STAT DATABASE

 

CAISS-STAT contains a built-in database of one-variable data sets that can be

used for exploration and analysis.
