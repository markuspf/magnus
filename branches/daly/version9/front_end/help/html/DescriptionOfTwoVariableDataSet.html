TWO VARIABLE DATA SETS

 

          A two variable data set consists of two collections of numbers.  These can either be independent or paired and generally represent  set of measurements such as heights, weights, time, stress etc. from two populations. A typical example would be

 

X       3.4 ,5.6, 7.9, 8.8

Y          4.6,5.6,4.8,8.3

 

Analysis of a two variable data set differs whether they are considered independent populations or correlated populations. If the sets of numbers are considered as samples from independent populations the two samples can be different sizes but if they are consider as samples from correlated populations they must be paired and the sample sizes must be the same.  The tools available for use with a two variable data set are subdivided into tools for independent populations and regression analysis tool for correlated populations.  These include

 

 

     Independent populations

Full Data Analysis

Data Summaries

Graphical Analysis

Goodness of Fit

Comparison of Variances

   Regression Analysis

            Full Regression Analysis

For Independent Populations :

 

The Data Summary Tool will construct a frequency distribution for each of the two given data sets.

 

The Full Data Analysis Tool will calculate all the basic descriptive measures for each of the two given data sets.

 

The Graphical Analysis Tool will construct various graphical presentations for each of the two given data sets.

 

The Comparison of Means Tool will calculate either a confidence interval for the difference of the two population means or evaluate a hypothesis test for the difference of the two population means against a given target difference.

 

The Comparison of Variances Tool will evaluate a hypothesis test for equality of the two population variances.

 

The Goodness of Fit Tool will test each of the two given data sets to see how well they fits various theoretical populations including a normal, exponential and uniform.

 

The Non-parametric test tool  will compare the given populations using the non-parametric Kruskal-Wallis Test

 

For Regression Analysis:

 

The Full Regression Analysis Tool will compute the standard regression diagnostics including the regression parameters, the coefficient of determination and the correlation coefficient.

 

The Graphical Analysis Tool will plot the given data points and then superimpose the fitted regression line. 

 

The Prediction and Parameter Testing Tool will allow the user to do various tests concerning the regression slope and regression intercept.  It will also allow the user to construct prediction intervals based on the fitted regression model.  Finally it will allow the user to do a test of model fit. 

 

The Residual Analysis Tool will construct a graph of the sample residuals and test the mean against zero and the fit against a normal model.

 

The Alternate Models Tool will allow the user to construct various alternative one variable models.  These include exponential and power models.  The fitted coefficient of determination will be compared to that of the fitted linear model.

 

The Correlation Analysis Tool  will compute the correlation coefficient and then allow the user to construct either a confidence interval for the correlation coefficient or conduct a hypothesis test concerning the correlation coefficient.
