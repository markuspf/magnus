DESCRIPTION OF DATA SET TYPES

 

MAGNUSSTAT allows the user to enter and then analyze and explore several different types of data sets.  These include

 

One Variable Data Set

Two Variable Data Set

Multi Variable Data Set

Attribute Data Set

Time Series Data Set

Quality Control Data Set

Probability Distribution

 

Below we give a description of each of these options.  Different tools are provided with each of the data set type options. 

 

(1)   ONE VARIABLE DATA SETS

 

          A one variable data set is a collection of numbers.  These numbers generally represent a set of measurements such as heights, weights, time, stress etc.  These will usually consist of a random sample from some continuous population.  A typical example would be

4.8, 8.9, 3.7, 6.6, 5.9

 

Analysis of a one variable data set usually includes determining the descriptive measures of the data, such as the mean and standard deviation and then using these calculations to infer population measures.  The tools available for use with a one variable data set include

 

Data Summary

Full Data Analysis

Analysis of the Mean

Analysis of the Variance

Goodness of Fit

Graphical Analysis

 

The Data Summary tool will construct a frequency distribution of the given data set. 

 

The Full Data Analysis Tool will calculate all the basic descriptive measures for the given data set.

 

The Graphical Analysis Tool will construct various graphical presentations of the given data set.

 

 

The Analysis of the Mean Tool will calculate either a confidence interval for the population mean or evaluate a hypothesis test against a given target mean.

 

The Analysis of the Variance Tool will calculate either a confidence interval for either the population variance or population standard deviation or evaluate a hypothesis test against a given target variance.

 

The Goodness of Fit Tool will test the data set to see how well it fits various theoretical populations including a normal, exponential and gamma.

 

 

(2)   TWO VARIABLE DATA SETS

 

          A two variable data set consists of two collections of numbers.  These can either be independent or paired and generally represent  set of measurements such as heights, weights, time, stress etc. from two populations. A typical example would be

 

X       3.4 ,5.6, 7.9, 8.8

Y          4.6,5.6,4.8,8.3

 

Analysis of a two variable data set differs whether they are considered independent populations or correlated populations. If the sets of numbers are considered as samples from independent populations the two samples can be different sizes but if they are consider as samples from correlated populations they must be paired and the sample sizes must be the same.  The tools available for use with a two variable data set are subdivided into tools for independent populations and regression analysis tool for correlated populations.  These include

 

 

     Independent populations

Full Data Analysis

Data Summaries

Graphical Analysis

Goodness of Fit

Comparison of Means

Comparison of Variances

Non-parametric Tests

Report

   Regression Analysis

            Full Regression Analysis

Graphical Analysis

Prediction and Parameter Testing

Residual Analysis

Alternate Models

Correlation Analysis

Report

 

For Independent Populations :

 

The Data Summary Tool will construct a frequency distribution for each of the two given data sets.

 

The Full Data Analysis Tool will calculate all the basic descriptive measures for each of the two given data sets.

 

The Graphical Analysis Tool will construct various graphical presentations for each of the two given data sets.

 

The Comparison of Means Tool will calculate either a confidence interval for the difference of the two population means or evaluate a hypothesis test for the difference of the two population means against a given target difference.

 

The Comparison of Variances Tool will evaluate a hypothesis test for equality of the two population variances.

 

The Goodness of Fit Tool will test each of the two given data sets to see how well they fits various theoretical populations including a normal, exponential and uniform.

 

The Non-parametric test tool  will compare the given populations using the non-parametric Kruskal-Wallis Test

 

For Regression Analysis:

 

The Full Regression Analysis Tool will compute the standard regression diagnostics including the regression parameters, the coefficient of determination and the correlation coefficient.

 

The Graphical Analysis Tool will plot the given data points and then superimpose the fitted regression line. 

 

The Prediction and Parameter Testing Tool will allow the user to do various tests concerning the regression slope and regression intercept.  It will also allow the user to construct prediction intervals based on the fitted regression model.  Finally it will allow the user to do a test of model fit. 

 

The Residual Analysis Tool will construct a graph of the sample residuals and test the mean against zero and the fit against a normal model.

 

The Alternate Models Tool will allow the user to construct various alternative one variable models.  These include exponential and power models.  The fitted coefficient of determination will be compared to that of the fitted linear model.

 

The Correlation Analysis Tool  will compute the correlation coefficient and then allow the user to construct either a confidence interval for the correlation coefficient or conduct a hypothesis test concerning the correlation coefficient.

 

(3)   ATTRIBUTE DATA SETS

 

        An attribute data set consists of data that is in the form of counts or frequencies.  Questions of interest for attribute data sets generally revolve around proportions, that is the theoretical proportion of some characteristic or attribute.  The entries in an attribute data set will consist of observed frequencies of characteristics or attributes.

For example suppose an election is being analyzed. A poll was conducted for Candidate A and the results were that out of 200 voters polled 80 would vote for Candidate A.  The attribute data would be 80 for Candidate A out of a total of 200.  Attribute data could also be cross-classified data in the form of a contingency table. 

       Attribute data sets can be used for simple proportion testing either with one sample or two; contingency table analysis or for general goodness of fit testing.

The check in for an attribute data set can be done in two different ways – for proportion testing or for contingency table analysis.

      For proportion testing what will be checked in is the proportion of a given characteristic in one or two samples.  The tools then available will be

 

The Confidence Interval Tool will calculate a confidence interval for a population proportion for each of the given samples

 

The Proportion Testing Tool will evaluate a hypothesis test for either a single sample against a given proportion or a comparison of the proportions in two different samples.

 

An r by k contingency table is a cross-classification table with r rows and k columns.  The entries are the frequencies in that cross-classification.  For example

 

 

B1

B2

B3

A1

86

57

91

A2

37

31

45

 

would be a 2 by 3 contingency table.  The entry 86 indicates an observed frequency of 86 in the A1-B1 cross-classification. The question of interest is whether the A classification is independent of the B classification or equivalently whether the proportion breakdown in each B classification is the same relative to the A classification.

The tools available with contingency table data include

 

The Table Analysis Tool will evaluate a chi-square value for a given contingency table.

 

The Full Analysis Tool will allow the user to interactively consider sub-tables of a given contingency table.
